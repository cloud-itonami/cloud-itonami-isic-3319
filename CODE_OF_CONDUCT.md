# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Participants should communicate respectfully, assume good intent when
possible, and focus disagreement on facts, design, safety and community
impact.

Unacceptable behavior includes harassment, threats, discrimination,
publishing private information, or using the project to expose real
equipment/work-order, customer or personal data.

## Enforcement

Maintainers may remove comments, reject contributions or restrict
participation when behavior puts contributors, repair technicians,
operators or the surrounding community at risk -- including proposing to
extend this actor's authority into repair-equipment/diagnostic-tool
control or return-to-service sign-off, which is out of scope for this
project by design.

Report conduct concerns privately to the maintainers through the
cloud-itonami organization.
